##  Sobre o Projeto

O **Contrato** é uma ferramenta desenvolvida para automatizar o fluxo de criação, revisão e armazenamento de contratos. O objetivo principal é reduzir erros manuais e garantir que os termos estejam sempre atualizados conforme os padrões da organização.

> [!IMPORTANT]
> Este projeto está em fase de desenvolvimento ativo. Sugestões são bem-vindas via Issues.

##  Funcionalidades Principais

- [x] **Templates Dinâmicos:** Variáveis que preenchem dados automaticamente.
